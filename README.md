#Customer Analytics Dashboard (Excel)
## Project Overview
This project is an interactive Customer Analytics Dashboard built in Microsoft Excel using Power Pivot, Pivot Charts, DAX Measures, and Slicers.
The dashboard focuses on analyzing:
•	Customer growth trends
•	Repeat customer behaviour
•	Revenue contribution
•	Profit
## Tools and features used
-Microsoft excel
-Pivot Tables
-Pivot Charts
-DAX measures
-Slicers
-Calculated Fields
-KPI Cards
-Interactive Dashboard Design
## Key KPIs
-Unique Customers
-Repeat Customer %
-Average Revenue per customer
-Average Order Value
-Profit Margin%
-Avg Discount%
## Key Insights
-Customer acquisition declines after 2017.
-Existing customer showed strong repeat behaviour (98%)
-Higher discounts reduced profitability across several sub-categories.
-Consumer segment generated highest revenue contribution,
-Customer loyalty remined strong despite slower growth.
##Dashboard features
-Interactive filtering using slicers
-Timeline based trend analysis
-Customer growth trend visualization
-Repeat customer distribution 
-Top customer contribution in sales and profit
-sales and profit margin% distribution of customer segment
-Profit vs discount behavior across sub categories of product
##Data Preparation
-Cleaned inconsistent date formats
-Created DAX measures for KPI analysis
-Organized pivot tables for dashboard interaction
